## ğŸ“ Dataset Information (hotel_bookings sheet)
The dataset contains detailed booking information with the following fields:

- **hotel** (City Hotel / Resort Hotel)  
- **is_canceled** (0 = Not Cancelled, 1 = Cancelled)  
- **arrival_date_year & arrival_date_month**  
- **adults, children, babies**  
- **country** (Guest nationality)  
- **reserved_room_type vs assigned_room_type**  
- **reservation_status & reservation_status_date**  
- **room_status** (Desired / Undesired)  
- **Guest_type** (Family, Single, Couples, etc.)  

This provides a complete picture of customer and booking behavior.
